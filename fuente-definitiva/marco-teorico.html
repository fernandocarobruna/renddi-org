05 / ECUACIONES ESTRUCTURALES

Modelo SEM (lavaan)

Ecuaciones estructurales que definen cómo cada variable es función de sus predecesores causales, más un error estocástico.

mastery = −0.30·anxiety + +0.20·confidence + +0.30·attention_span + +0.30·study_adherence + +0.30·self_efficacy + ε₁

attention_span = +0.45·sleep_quality + −0.40·fatigue + −0.30·anxiety + ε₂

anxiety = +0.45·stress_level + −0.35·self_efficacy + ε₃

confidence = +0.35·self_efficacy + ε₄

emotional_state = +0.30·physical_activity + +0.30·social_support + +0.25·parental_support + −0.25·fatigue + −0.35·stress_level + ε₅

self_efficacy = +0.30·mastery(t-1) + ε₆

motivation = +0.30·emotional_state + +0.30·vocational_clarity + ε₇

study_adherence = +0.35·motivation + +0.25·metacognition + +0.20·frustration_tolerance + +0.30·time_management + ε₈

sleep_quality = −0.40·screen_time + +0.25·physical_activity + −0.35·stress_level + ε₉

fatigue = +0.30·cognitive_load + −0.50·sleep_quality + ε₁₀

frustration_tolerance = +0.25·growth_mindset + ε₁₁

Resolución del Ciclo mastery ↔ self_efficacy

El ciclo mastery → self_efficacy → confidence → mastery se resuelve temporalmente:
mastery(t) → self_efficacy(t) → confidence(t) → mastery(t+1)

Ganancia del ciclo: G = 0.30 × 0.35 × 0.20 = 0.021 (converge rápidamente, G·γ³ = 0.007 ≪ 1)

Convergencia y Estabilidad

Pruebas formales de que el algoritmo de propagación causal converge y el sistema es robusto ante perturbaciones de pesos.

δ(k+1)[j] = Σ(i,j)∈E δ(k)[i] · wij · γk
